# Script-de-gestion-des-eleves-sur-un-Proxmox-pedagogique
Script interactif tout-en-un pour gérer facilement les comptes élèves sur un serveur Proxmox VE utilisé dans un contexte pédagogique.  Conçu par des formateurs pour des formateurs, ce script automatise la création, la suppression et la gestion des droits des comptes élèves, tout en garantissant la sécurité et la simplicité d'utilisation.
